# PairProjectTemplate
In hw-four there is a short game, created collaboratively. It has two side rooms that you need to go through before you can enter the final room. This project was created so that we could solidify our skills in global variables and in interactables. I made some assets, the actual things that are interacted with, the room layouts, the level and sound design, and deciding the different types of interactables. I learned how to make global variables and how to reliably make an interactable node that can be individually changed depending on what you need. I also learned how to work Github in a collaborative way. If I could do this project again, I would spend more time figuring out how to keep levels as they are when you leave them, instead of just reloading them upon re-entry. I would also spend more time personally on assets, as that is what I find the most fun.
